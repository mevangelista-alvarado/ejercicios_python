# Ejercicio 23
_____

__Área y perímetro de un rectángulo__  

_____

Escribir una función que dado el largo y ancho de un rectángulo calcule y muestre en consola el área y el perímetro del rectangulo.

**Notas**  

No utilice módulos de `Python`.
